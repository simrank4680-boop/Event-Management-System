# Smart Event Management System

##Overview
This project is a simple Java-based Smart Event Management System. It demonstrates basic Object-Oriented Programming (OOP) concepts such as classes, objects, and methods.

The system allows:
- Creating an event
- Adding participants
- Displaying event details


## Technologies Used
- Java
- GitHub
 
-## Project Structure
- Main.java → Contains all classes and main method



## How to Run
1. Open the project in any Java IDE (IntelliJ / Eclipse)
2. Run the Main.java file
3. View output in console
